# Students Academic Analysis Engine

A full-stack web application built using Java Spring Boot and MySQL to analyze student academic performance and generate meaningful academic insights.

---

## 🚀 Project Overview

The Students Academic Analysis Engine is designed to help institutions monitor, analyze, and evaluate student academic performance across multiple semesters and subjects.

The system provides performance tracking, weak subject identification, and semester-wise academic trend analysis.

---

## 🛠️ Tech Stack

Backend:
- Java 17
- Spring Boot
- Spring Data JPA
- REST APIs

Frontend:
- HTML
- CSS
- JavaScript

Database:
- MySQL

Tools:
- Eclipse
- Postman
- Git & GitHub

---

## 🧩 Features

- User authentication (Admin, Faculty, Student)
- Student academic data management
- Subject-wise marks storage
- Semester-wise performance tracking
- Academic analysis module
- Weak student identification
- Academic report generation

---

## 🏗️ Architecture

The application follows a layered architecture:

Controller Layer – Handles HTTP requests  
Service Layer – Contains business logic  
Repository Layer – Handles database operations using JPA  

This structure ensures separation of concerns and maintainability.

---

## 📊 Academic Analysis Logic

The system performs:

- Average score calculation
- Subject performance analysis
- Semester performance trend tracking
- Identification of weak subjects based on predefined threshold
